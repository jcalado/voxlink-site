# Primeira Ligação

Configure um perfil de servidor, ligue-se a um refletor e faça a sua primeira transmissão.

![Formulário Adicionar Servidor](/docs/add-server.webp)

## Adicionar um Perfil de Servidor

1. Abra o VoxLink e toque no separador **Servidores**.
2. Toque em **+** para criar um novo perfil.
3. Preencha os seguintes campos:
   - **Nome** — uma etiqueta para este perfil (ex.: `O Meu Repetidor`)
   - **Hostname** — o endereço do refletor (ex.: `vps.ngen.org`)
   - **Porta** — a porta do refletor, normalmente `5300`
   - **Talkgroup** — o talkgroup a que se quer juntar (ex.: `999`)
   - **Indicativo** — o seu indicativo como aparecerá para outras estações
   - **Chave de Autenticação** — a chave de autenticação fornecida pelo administrador do refletor
4. Toque em **Guardar**.

> Verifique a chave de autenticação — é sensível a maiúsculas/minúsculas e é a razão mais comum para uma primeira ligação falhada.

## Ligar

1. No separador **Servidores**, toque no perfil que acabou de criar.
2. O indicador de estado muda para **A ligar…** e depois **Ligado**.
3. O ecrã PTT mostra o seu talkgroup e quaisquer nós ligados.

![Ecrã PTT](/docs/ptt-screen.webp)

## Fazer a Sua Primeira Transmissão

1. Toque no separador **PTT**.
2. O VoxLink usa o modo **Latch** por defeito — toque no botão **PTT** para começar a transmitir, toque novamente para parar.
3. O medidor **TX** mostra o nível do seu microfone. Aponte para picos entre −10 dB e −6 dB.
4. Quando outra estação transmite, o medidor **RX** ativa-se e o indicativo do operador (ex.: `CT2GNH-M`), nome e localização aparecem acima dele.

> Na sua primeira transmissão, anuncie o seu indicativo e confirme que consegue ouvir outras estações. Isto verifica que tanto TX como RX estão a funcionar.
