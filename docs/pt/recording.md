# Gravação

Grave áudio do talkgroup e reproduza mais tarde. Ative a gravação em **Definições > Gravação**.

## Iniciar uma Gravação

Uma vez ativada, os controlos de gravação aparecem no canto superior direito do ecrã PTT enquanto está ligado a um servidor.

Toque no botão de **gravação** para iniciar. Um temporizador vermelho mostra o tempo decorrido no formato `MM:SS`. Toque no botão novamente para parar. A gravação é guardada automaticamente.

As gravações capturam o áudio do talkgroup como o ouve — tanto transmissões recebidas como enviadas.

## Navegar nas Gravações

Toque no ícone de **pasta** junto ao botão de gravação para abrir o ecrã de Gravações. Cada gravação mostra:

- **Data e hora** da gravação
- Número do **Talkgroup** (com etiqueta do preset se configurada)
- **Duração** em minutos e segundos
- **Tamanho do ficheiro**

As gravações são ordenadas da mais recente para a mais antiga.

## Reprodução

Toque numa gravação para reproduzi-la. Uma barra de progresso aparece abaixo do cartão mostrando a posição atual. Toque novamente para parar a reprodução. Apenas uma gravação pode ser reproduzida de cada vez.

## Partilha

Cada gravação tem um botão de **partilha** que abre o menu de partilha padrão do Android. Envie gravações por email, aplicações de mensagens, armazenamento na nuvem ou qualquer outro destino de partilha no seu dispositivo.

As gravações são guardadas como ficheiros WAV padrão (16 kHz, 16-bit PCM mono) — compatíveis com qualquer leitor ou editor de áudio.

## Detalhes do Ficheiro

| Detalhe | Valor |
|---|---|
| **Formato** | WAV (PCM não comprimido) |
| **Taxa de amostragem** | 16 kHz |
| **Profundidade de bits** | 16-bit |
| **Canais** | Mono |
| **Tamanho aprox.** | ~2 MB por minuto |
| **Nome do ficheiro** | `TG{número}_{AAAAMMDD_HHMMSS}.wav` |

As gravações são armazenadas no diretório privado da aplicação VoxLink. Não são visíveis na aplicação Ficheiros do Android — use o ecrã de Gravações na aplicação para as gerir.

> Para libertar espaço, deslize uma gravação para a eliminar. Gravações eliminadas não podem ser recuperadas.
